## 🎯 **Mono设计原则**

### 色彩体系 🎨
```css
/* 主色调 - 灰色系 */
gray-50   /* 页面背景 */
gray-100  /* 卡片背景 */
gray-200  /* 边框分割 */
gray-400  /* 图标占位 */
gray-600  /* 图标主色 */
gray-900  /* 主要文字 */

/* 强调色 - 语义色彩 */
green-100, green-800  /* 成功状态 */
red-100, red-800      /* 错误状态 */
blue-100, blue-800    /* 信息提示 */
```

### 间距系统 📏
```css
/* 标准间距 */
space-y-6   /* 主要区块间距 */
gap-6       /* 网格间距 */
p-4, p-6    /* 卡片内边距 */
```

### 圆角系统 🔲
```css
rounded-lg  /* 主要卡片圆角 */
rounded-md  /* 小元素圆角 */
```

### 阴影系统 ✨
```css
hover:shadow-md     /* 悬停阴影 */
shadow-sm          /* 微妙阴影 */
backdrop-blur-md   /* 毛玻璃效果 */
```
